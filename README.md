# TasksAPI-demo
Flask-RESTful API

Pre-Requisites
-----
Install Python 3, Virtualenv.




Linux, OS X, Cygwin
--------------
    chmod +x /path/to/setup.sh
    Run `./setup.sh`
	

Windows
--------------
    `setup.bat` (Windows)

RUN
---------
Server:
/venv/bin/python rest-server.py

ACCESS
--------
Client:
http://localhost:5000/index.html` 




MORE INFO:

	setup.sh:
	---------
	virtualenv venv
	pip install -r requirements.txt
