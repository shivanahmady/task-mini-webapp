# TasksAPI-demo
Flask-RESTful API

Pre-Requisites: Install Python 3, Virtualenv.



Linux, OS X, Cygwin
--------------
`chmod +x setup.sh`
`./setup.sh`
	

Windows
--------------
`setup.bat` 

Start Server
---------
`/venv/bin/python rest-server.py`

Access Client
--------
http://localhost:5000/index.html`
