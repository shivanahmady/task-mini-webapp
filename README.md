# TasksAPI-demo
Flask-RESTful API

Pre-Requisites
-----
Install Python 3, Virtualenv.


Linux, OS X, Cygwin
--------------
    chmod +x /path/to/setup.sh
    Run `./setup.sh`

Windows
--------------
    `setup.bat` (Windows)

RUN
---------
Server:
/venv/bin/python rest-server.py
